Face And Lips V0.3.8.1
2013/01/08

MMD表情モーション作成を補助するツールです。
グラフィカルに表情を編集することができ、表情モーションをVMDファイルとして出力することができます。

MP3, AVIなどの様々なファイルも再生できます。
DirectShowを使用したWindows Media Playerのコーデックで動いておりますので、MP4やFLVといったファイルは、追加コーデックが必要となります。

■必要環境
========================================================
Windows XP 以降
.NET Framework 2.0以降
Microsoft Visual C++ ランタイムコンポーネント
DirectX 9.0c (2010 Feb以降)

■インストール/アンインストール方法
========================================================
解凍したファイルを任意のフォルダに置くだけです。
レジストリなどに書き込みはしておりません。
アンインストールする場合はフォルダごと消してください。


■大まかな使い方流れ
========================================================
1. FaceAndLips.exeを起動します
2. 上部メニュー「PMDをインポート」から、モデル(PMD)ファイルを読み込みます。
3. 上部メニュー「Waveをインポート」から、メディアファイルを読み込みます。
4. 中ほどにある「あ」「まばたき」などの表情ボタンを下のトラックにドラッグ&ドロップして
　表情を置いていきます。
5. 「再生」ボタンを押すと、表情を確認することができます。
6. 上部メニュー「VMDの書き出し」から、表情モーションファイルを出力します。

* 読み込めるファイルは、Windows Media Playerで再生可能なファイルです。
　必要であればWindows Media Playerのコーデックをインストールしてください。
* 頭ボーンのないモデルは読み込めません。

*マニュアルはこちら
 https://sites.google.com/site/moggproject/home/faceandlips
